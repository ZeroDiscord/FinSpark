# Demo Script

## Best 3-5 Minute Demo Flow
1. Open landing page
   Say: "This platform helps enterprise teams understand exactly which product feature is driving churn."
2. Open workspace selection
   Say: "Each tenant is isolated, so every bank gets its own analytics and recommendations."
3. Upload APK or website input
   Say: "We start by detecting features directly from enterprise software."
4. Show detected features
   Say: "Instead of starting with raw events, we first understand the product structure."
5. Open tracking SDK page
   Say: "Then we generate instrumentation so product teams can start collecting usage immediately."
6. Show dashboard
   Say: "Now we convert raw usage into adoption, churn, funnel, and journey intelligence."
7. Highlight churn hotspot
   Say: "Here, Credit Check is clearly the biggest drop-off step."
8. Open recommendation center
   Say: "The system doesn’t stop at analytics. It generates prioritized product actions."
9. Send recommendation to Asana
   Say: "Teams can push a recommendation directly into their delivery workflow."
10. Export to Power BI
   Say: "And if the enterprise needs centralized BI, the same data can be exported instantly."

## Demo Close
Say: "We’ve connected feature detection, tracking, prediction, analytics, and actioning into one enterprise workflow."
